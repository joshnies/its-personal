# Personal Website

Static site for hosting all of my articles, open-source projects, and (soon) UI/UX designs.

Built with [Astro](https://astro.build).

### Prerequisites

| Name | Version |
| ---- | ------- |
| bun  | 1.x     |

```sh
# Install dependencies
bun install

# Build (production)
bun run build

# Run (local)
bun run dev

# Preview build
bun run preview
```
